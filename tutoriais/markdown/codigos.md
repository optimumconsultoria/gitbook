# Códigos

Os códigos podem ser adicionados em linha junto ao texto entre dois acentos graves ( \` ):

```markdown
Existe um `código` nessa linha
```

A criação de um bloco de códigos é feita com três acentos graves ou três tils ( \~ ):

````markdown
```
def f(x):
    return x ** 2
```
````

```python
def f(x)
    return x ** 2
```
