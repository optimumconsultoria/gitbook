# Tabelas

As tabelas em Markdown são visualmente intuitivas, utilizamos " | " para separar colunas e " - " para separação de linhas:

```markdown
Coluna 1 | Coluna 2
---------|----------
Valor 1  | Valor 2
Valor 3  | Valor 4
Valor 5  | Valor 6
```

| Coluna 1 | Coluna 2 |
| -------- | -------- |
| Valor 1  | Valor 2  |
| Valor 3  | Valor 4  |
| Valor 5  | Valor 6  |
